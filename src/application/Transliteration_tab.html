"Transliterate" Tab

In this tab you can transliterate the your own input text. The transliterated text outputs in the text box bellow.

The transliterated text is editable. Edits to the transliterated text will not affect further transliterations. To transliterate the text from result text box you need to enter it in the input text box.

Button "Voynich/Latin"

Use this button to change the font of your text and further input.This will change the only the font of the text you have entered. You can flip back and forth from fonts and you will not lose the content.

  • The button "Latin" changes your texts input font to a latin type font.
  • The button "Voynich" changes the Latin font to a manuscript

Button "Transliterate"

This button transliterates the text of you input according the Transliteration table of your choice. The transliterated text appears in the text box bellow input text box.
