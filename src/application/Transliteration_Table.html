Transliteration Table

Transliteration tables are used as rules by which the text gets transliterated. Before performing transliteration you must choose the preffered transiliteration table.

Click on the ▼ symbol to open available table list.

There are 5 preloaded tables:

  • "BasicEVA_to_ASCIIsounds"
  • "Bennett"
  • "Bennett_to_FSG"
  • "Currier"
  • "FSG"

These tables can not be deleted and/or changed.

Button "Add"

Use this to open a new windows in which you can create new custom transliteration tables.

Adding a new transliteration table

To add a new custom table:

  1. Write the new tables name in field "New table name:"
  2. Choose one of the available tables from list as a template by clicking on the ▼ symbol
  3. Edit table as wanted in text box. Text must be in for "FROM=TO", where FROM is the input character/-s and TO is the character/-s it must change into.
  4. Press "Save" to save the new table
    1. After this continue adding new tables using the previous steps or close the window by clicking "x"
  5. Press "Cancel" to cancel the operation and close window.

Button "Delete"

Click this to delete the currently chosen transliteration table. After deleting, the table will not be reversible.

Button "Refresh"

Click this button to refresh the table choice list and to add recently created tables to it.
