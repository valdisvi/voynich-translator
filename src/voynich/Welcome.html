Welcome to User Guide

This is a Voynich Manuscript transliteration application.

Using this you can simplify and automate the transliteration process by quickly getting the transcription of the page and applying different table transliterations to text generated from your own input or an online source. You can create your own transliteration tables and apply their rules to the generated text, as well as use the provided default tables. 

Choose one from the available topics you are interested in on the windows left side and click on it to read.

Available topics are:

  • Transliteration Tab: instructions on how to use functions in tab called "Transliteration"
  • Web tab: instructions on how to use the functions in tab called "Web" and information about the available transcriptions
  • <Transliteration table: information about the transliteration tables and instructions on how to add and/or delete them
