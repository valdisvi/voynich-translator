"Web" Tab

This tab is used to get input manuscript text from a web source.
The web source currently used is http://www.voynich.nu/

On this web source, Voynich manuscript pages are collected in quires that contain varying amount of pages in them. Pages are marked with a number and a layout assigning letter r or v. The pages also have several differing transcriptions that can be discerned by their code symbol/letter.

Possible issues with source text

Please note that:

  • Some pages are missing from queries or are not transliterated;
  • Some page naming is inconsistent, therefore, cannot be properly loaded;
  • Not all transcription versions are available for all the pages;
  • Several pages contain special untranscribed symbols (usually identifiable by inclusion in curly brackets {} ).

In case of an issue with getting source text, a popup window explains the cause of this problem.

Please refer to the website (http://www.voynich.nu/)  for additional context and source insight.

Navigate between pages

Page number

To change the page number, you can use buttons Next and Previous in the lower left corner of the frame to navigate to and load the page, or input the page number manually and then click on Get Text button to load it.

Letter

To change the layout assigning letter, set the necessary value (r or v) in the lower left corner of the frame. To refresh/change input text according to selection, click on Get Text button.

Set transcription text

Transcription version

To set the transcription version, use the selection box and select the transcription source, or type the transcription version code symbol into box field. To load your selected transcription, use the "Get Text" button. If necessary, you can edit the contents of loaded text. Your edits will be used in transliteration.

The link to currently used source file can be seen in the Source: text box in the lower right corner.

Button "Get Text"

Click on this button to load text with currently set parameters into input text field.

Button "Delete"

Click on this button to clear the input text field.

Button "Transliterate"

Click on this button to transliterate the text in input text field according to the chosen transliteration table.

Button "◄" and Button "►"

Clicking on these buttons changes the page number to the previous (◄) or next (►) page.
